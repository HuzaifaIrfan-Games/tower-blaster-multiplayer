# 🚀 Usage

### Gameplay
Build the Tower in Accending order before your Opponent


### Installing Dependencies
```bash
pip install -r requirements.txt
```

### Running the Server and Web GUI Client
```bash
python server.py
```

### Running the (CLI/TUI) Client
```bash
python tui-client/client.py
```
